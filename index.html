Firefox-Muter by yxl

A firefox addon to mute the browser

Currently the addon works on windows and Unbuntu. It could mute firefox on Win7/Vista/XP and so on. Linux systems which use PalseAudio to manage their audio are supported as well, but we have tested Ubuntu 10.10 with Firefox 9 and 3.6 only. Mac will be supported later.

We are two Chinese guys and create this addon just for fun in our spare time. There is not much time for us to test and decorate the addon. To push forward the project, it's in need of your help.

Known issues(已知问题)

The anti-virus softwares of AVG and AVAST will block the addon's function mistakenly, as we use "Hook" techonology which is also used by some viruses. In order to let the addon work properly, you can add "MuterHook-32.dll" to the white list of your anti-virus software.(This problem has been solved in version 9.0.3)

一些杀毒软件(AVG和AVAST)会错误地让这个附加组件失效。 因为我们使用了"钩子"技术，这种技术被一些病毒所使用。为了让这个附加组件正常工作，可以把"MuterHook-32.dll"添加到杀毒软件的白名单中。(这个问题已经在0.9.3版中解决)

Install

Firefox-Muter is a standard Firefox addon. Download the addon file muter-0.9.3.xpi and open with Firefox to install it.
