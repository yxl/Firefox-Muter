These examples are used in development to test Firefox-Muter and should allow you to quickly experiment with the extension.Most of them are from mutetab(http://www.mutetab.com/mutetabdoc.html) project. Please let me know (xyuan at mozilla.com) if you run across any audio source (regardless of if it is on this list) that does not detect or perform operations correctly, and I'll add it as a test case and see if I can make a fix.
